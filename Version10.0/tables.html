How To Properly Code In Tables In 7 Steps

Step 1: Start your webpage with <!DOCTYPE html>
  • This will tell your browser that this is the latest version of the HTML. This will help your browser organize everything
    •
    Step 2: Create the tag element in the second line <html>
  • This tag element will root the element to the HTML page and all the other elements except (DOCTYPE)
    •
    Step 3: After doing that then you will create style for your CSS <style> </style>
  • The style is used for CSS to style sheet elements in your webpage. For this case it is defining the style for tables. Once you have done that then you will paste table, th, td { border: 1px solid black; }
  • Inside the style CSS
    • Step 4: Once you have done that then you will need to add a body element underneath the style element provided below <body>
    Step 5: After that in the next line underneath body you will add this headers. This will create a heading for your webpage <h2>Example Of Code Working</h2>
    Step 6: You will create a table element by using this code putting it underneath the heading this will set you up for your list <table style="width:100%">
    Step 7: Then you will add the first rows with a table headings to represent what your talking about with this code (bolded) <tr> <th>Person 1</th> <th>Person 2</th> <th>Person 3</th> </tr>
    Step 8: You will add another set of rows. This time it will be for data that goes underneath the bolded text that you coded <tr> <td>Emil</td> <td>Tobias</td> <td>Linus</td> </tr> <tr> <td>16</td> <td>14</td> <td>10</td> </tr>
    Step 9: Finally you will need to add your closing tags which are table and html that I will provide below </table> </html>
  • Once you have completed all these steps in the correct order you should get yourself a table with bolded words and slots to put in data about anything you desire!!
