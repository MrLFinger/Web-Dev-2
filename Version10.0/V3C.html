How to use Box Model

Here are the four different models of box model

1. Content- The image, text, or other media content in the element 2. Padding- Space that's between the box's content, and Border. 3. Border- The line that is between the box's padding and margin. 4. Margin- The space between the box and it's surrounding boxes.

Padding Example
Padding example: This box has 20px padding. Code:
<meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Padding Example</title> <style> .box { width: 200px; height: 100px; background-color: #3498db; padding: 20px; color: white; font-size: 16px; border: 2px solid #2980b9; } </style> <div class="box"> Padding example: This box has 20px padding </div>
